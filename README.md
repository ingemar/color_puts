# ColorPuts

Color your output with inline color codes.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'color_puts'
```

## Usage

Example:

```ruby
  include ColorPuts

  cputs "There was a %green{field} and %blue{sky}."
```
